创建 `student_class.py` 文件。手工编写并运行以下代码，前半部分理解类与对象的定义、`__init__` 与 `__str__` 魔术方法，后半部分构建一个模拟“GeoAI 空间建筑物矢量要素面向对象封装与多实例数据计算系统”：  

```python
# -*- coding: utf-8 -*-
"""
程序名称：student_class.py
作者：黄乐源
日期：2026-07-11
描述：掌握面向对象编程基础、类与对象实例化、self机制、魔术方法与类属性，构建空间建筑物矢量要素类
"""

print("================= 任务 1：类与对象基础、魔术方法与实例方法 =================")

class SpatialBuilding:
    """
    GeoAI 空间建筑物矢量要素类
    用于封装单个建筑物斑块的几何参数、属性数据及空间运算行为
    """
    
    # 类属性：被所有 SpatialBuilding 实例共享（华中师范大学城市与环境科学学院 空间参考系坐标系统）
    crs_system: str = "EPSG:4326 (WGS84)"
    total_building_count: int = 0  # 类属性：用于累计全局创建的建筑物对象数量

    def __init__(self, building_id: str, footprint_area: float, height: float, is_active: bool = True):
        """
        魔术方法：构造函数，在对象实例化时被自动调用，用于初始化实例属性
        """
        self.building_id: str = building_id            # 实例属性：建筑物唯一编号
        self.footprint_area: float = footprint_area    # 实例属性：占地面积(平方米)
        self.height: float = height                    # 实例属性：建筑高度(米)
        self.is_active: bool = is_active              # 实例属性：是否参与提取分割计算
        
        # 每创建一个新实例对象，全局类属性计数器加 1
        SpatialBuilding.total_building_count += 1

    def calculate_volume(self) -> float:
        """
        实例方法：根据占地面积与高度，计算该建筑物的估算体量/体积(立方米)
        """
        # 通过 self 访问当前实例对象自身的属性
        volume = self.footprint_area * self.height
        return volume

    def update_footprint(self, new_area: float) -> None:
        """
        实例方法：修改当前建筑物的占地面积属性
        """
        print(f"🔧 [修改属性] 建筑物 {self.building_id} 面积更新: {self.footprint_area} m² -> {new_area} m²")
        self.footprint_area = new_area

    def __str__(self) -> str:
        """
        魔术方法：定义对象的打印字符串表示，当使用 print(obj) 时自动触发调用
        """
        status_str = "激活" if self.is_active else "冻结"
        return f"🏢 <SpatialBuilding ID={self.building_id} | 面积={self.footprint_area:.1f}m² | 高度={self.height:.1f}m | 状态={status_str}>"

# 实例化第一个建筑物对象 (华师城环科学会堂)
b1 = SpatialBuilding(building_id="CCNU_BUILDING_01", footprint_area=1250.5, height=24.0)

# 实例化第二个建筑物对象 (华师图书馆)
b2 = SpatialBuilding(building_id="CCNU_BUILDING_02", footprint_area=3800.0, height=32.5, is_active=True)

# 打印对象，触发 __str__() 魔术方法
print("📋 打印实例化后的建筑物对象：")
print(b1)
print(b2)

# 调用实例方法计算建筑物几何体量
v1 = b1.calculate_volume()
v2 = b2.calculate_volume()
print(f"📐 建筑物 {b1.building_id} 估算总体积: {v1:.2f} 立方米")
print(f"📐 建筑物 {b2.building_id} 估算总体积: {v2:.2f} 立方米")


print("\n================= 任务 2：类属性共享机制与实例属性修改操作 =================")

# 访问类属性（推荐使用类名直接访问，也可以使用实例对象访问）
print(f"🌐 统一空间参考系 (类属性): {SpatialBuilding.crs_system}")
print(f"📊 内存中已创建的建筑物要素总数 (类属性): {SpatialBuilding.total_building_count} 个")

# 通过实例方法修改属性
b1.update_footprint(1300.0)
print(f"🔄 修改后重新计算 {b1.building_id} 总体积: {b1.calculate_volume():.2f} 立方米")

print("\n🏁 Day14 面向对象类与对象定义、self机制及魔术方法实验成功交付！")
```

**77 类与对象**：

**面向过程编程**：其核心思想就是把一个需求分解成一系列要执行的步骤，然后按照步骤依次执行这些任务（关注的是**流程**、**步骤**）。面向过程编程非常简单，适合**简单、线性**的任务。

**对象**：可以理解为**现实中具体的人/物在程序中的数字化身**（**万物皆对象**）。它把一个人/物的**特征（称之为属性）**和**功能（称之为方法）**打包到一起，是**面向对象编程的基本单元**（关注的是谁来帮我做这件事）。

**类**：描述的是**一组具有相同属性（特征）和方法（功能/行为）的模板**。**对象**是**类的实例**，是**基于类创建**出来的（实例对象）。

提示：对象是由类创建出来的，**创建对象的过程**，也称为**对象的实例化**。一个类可以创建**无数个**对象。

**78 类与对象（进阶）**：

定义类的语法如下：

```python
# 定义类 ---> 不推荐 动态地为对象添加属性
class 类名:
    pass
    
# 创建对象
对象名 = 类名()
对象名.属性名1 = 属性值1
对象名.属性名2 = 属性值2

# 定义类 ---> 不推荐 动态地为对象添加属性
class Car:
	pass

# 创建对象
c1 = Car()
# 动态地为对象添加属性
c1.color = "red"
c1.brand = "BMW"
c1.name = "x5"
c1.price = 500000
print(c1.__dict__)


# 定义类 ---> 定义类时指定实例属性（推荐写法）
class 类名:
	def __init__(self, 参数列表):
		self.属性名 = 参数值
		self.属性名 = 参数值

# 创建对象
对象名 = 类名(参数列表)

# 定义类 ---> 定义类时指定实例属性（推荐写法）
class Car:
    def  __init__ (self, c_brand, c_name, c_price):
        self.brand = c_brand
        self.name = c_name
        self.price = c_price

# 创建对象
c1 = Car ("BMW","X5", 500000)
print(c1.__dict__)
```

**类名的命名规范**：遵循大驼峰命名法，**每个单词的首字母都是大写**，**单词之间没有分隔符**，比如:**U**ser**I**nfo，**U**ser**A**ccount。

__dict__：Python中用户自定义类实例的一个特殊属性，用于以**字典**形式**存储对象的属性**。

**定义在类的外面**称之为**函数**，**定义在类中的函数**称之为**方法**。

__init__：**初始化**方法，对象创建后**自动调用**，主要用于**设置对象的初始状态**（**设置对象属性**）。

**self**：类中定义的方法的第一个参数，表示**当前创建的实例对象**。

**79 实例方法**：

在类中定义实例方法时，定义语法与之前学习的函数定义的方式是一致的，如下：

```python
# 定义类
class 类名:
    def __init__(self，形参列表):
        self.属性名 = 参数值
        self.属性名 = 参数值
    def 方法名(self，形参列表):
    	...
    def 方法名(self，形参列表):
    	...

# 创建对象
对象名 = 类名(参数列表)
对象名.方法名(实参)

class Car:
    def _ init__(self, brand, name, price):
        self.brand = brand
        self.name = name
        self.price = price
    # 定义实例方法
    def running(self):
        print(f"{self.brand} {self.name} 正在高速行驶...")
    def total_cost(self, discount, rate):
        return self.price * discount + self.price * rate
    
# 创建对象
c1 = Car("BMW", "X5", 500000)

# 调用类中方法
total_cost = c1.total_cost(0.9, 0.1)
print(f"提车总价为:{total_cost:.0f}")

c1.running()
```

**80 魔法方法**：

**魔法方法**：指Python中提供的**以双下划线开头和结尾**的特殊方法，用于**定义类的特殊行为**，比如：__init__。

魔法方法**不需要我们手动调用**，Python会在合适的时机**自动调用**。

**81 类属性**：

属性分为：

**实例属性**：实例属性属于**每个具体对象的属性**，每个对象都是**独立**的（各个对象**特有**的数据）。在__init__方法中定义，通过**实例对象.属性**的方式操作。

**类属性**：类属性是属于**类本身的属性**，**所有实例共享**的（所有对象**共享**的数据或配置）。在__init__方法外定义（通常在类的定义开头），通过**类名.属性**的方式操作。
