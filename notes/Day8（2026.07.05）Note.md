```
# -*- coding: utf-8 -*-
"""
程序名称：geo_sample_manager.py
作者：黄乐源
日期：2026-07-05
描述：全面攻克列表核心方法、切片语法与案例全串联，实现遥感科研样本的动态录入、清洗与统计
"""

print("================= 任务 1：列表核心方法全演练（多视频案例综合） =================")
# 初始化一个华师大资环学院遥感解译小组的初始名单
research_team = ["张三", "李四", "王五"]
print(f"原小组成员: {research_team}")

# 1. 动态增加：append 末尾追加，insert 指定位置插入
research_team.append("赵六")
research_team.insert(1, "钱七")
print(f"增添成员后 (钱七加入索引1, 赵六加入末尾): {research_team}")

# 2. 批量合并：extend 引入外部合作团队
external_partners = ["周八", "吴九"]
research_team.extend(external_partners)
print(f"合并外部合作者后: {research_team}")

# 3. 动态删除：remove 按值删除，pop 按索引弹出
research_team.remove("张三")  # 张三毕业退出
graduated_member = research_team.pop(2) # 弹出索引为2的成员
print(f"毕业生退出后，当前团队成员: {research_team} (刚才弹出的成员是: {graduated_member})")

# 4. 检索与统计：index 找位置，count 计算出现次数
search_name = "王五"
if search_name in research_team:
    print(f"🔍 查询到 {search_name} 仍在组内，其当前索引位置为: {research_team.index(search_name)}")
print(f"📊 团队中名为 '王五' 的学者数量: {research_team.count('王五')}")


print("\n================= 任务 2：高频切片与空间降采样抽样模拟 =================")
# 模拟一段长序列的空间栅格像元点或者样点编号
spatial_samples = ["Grid_01", "Grid_02", "Grid_03", "Grid_04", "Grid_05", "Grid_06", "Grid_07", "Grid_08"]
print(f"原始格网样点序列: {spatial_samples}")

# 左右闭开切片
sub_samples = spatial_samples[1:5]
print(f"切片 [1:5] 截取中间段: {sub_samples}")

# 步长抽样（模拟遥感影像分辨率降采样，每隔一个像元抽样一次）
downsampled = spatial_samples[::2]
print(f"步长为 2 降采样抽样 [::2]: {downsampled}")

# 逆序反转
reversed_samples = spatial_samples[::-1]
print(f"利用切片实现序列完全反转 [::-1]: {reversed_samples}")


print("\n================= 任务 3：多案例融合终极实战（遥感解译样点数据清洗系统） =================")
# 场景：模拟从野外采集回来的建筑物、水体等遥感解译样本的置信度评分（0-100分）
# 目标：通过循环录入，剔除不合格样本，最终利用列表内置函数算出样本优质率和平均分
sample_scores = []
print("🛰️ 欢迎进入刘鹏程老师课题组：地物解译样本质量动态审计中心")
print("💡 提示：请输入每批次遥感样点的置信度得分（0-100），输入 -1 结束录入。")

while True:
    input_val = input("请输入样点置信度得分: ").strip()
    
    score = float(input_val)
    if score == -1:
        print("📥 录入终止，开始启动样本容器数据清洗...")
        break
        
    if 0 <= score <= 100:
        sample_scores.append(score)
        print(f"✅ 样点成功压入容器，当前已捕获样本数: {len(sample_scores)}")
    else:
        print("⚠️ 异常：置信度必须在 0 到 100 之间，请重新检查数据！")

# 综合数据处理（融合案例3思维）
if len(sample_scores) > 0:
    total_samples = len(sample_scores)
    max_score = max(sample_scores)
    min_score = min(sample_scores)
    avg_score = sum(sample_scores) / total_samples
    
    # 过滤出置信度优秀（>=85分）的科研核心样点
    excellent_samples = []
    for s in sample_scores:
        if s >= 85.0:
            excellent_samples.append(s)
            
    print("\n--- 📊 华师大资环学院 GeoAI 样本审计汇报 ---")
    print(f"1. 成功捕获样点总数: {total_samples} 个")
    print(f"2. 样点最高置信度: {max_score:.2f} | 最低置信度: {min_score:.2f}")
    print(f"3. 整体样本平均置信度: {avg_score:.2f}")
    print(f"4. 置信度优秀样点集(>=85): {excellent_samples}")
    print(f"5. 科研核心优质样本占比: {(len(excellent_samples)/total_samples)*100:.1f}%")
else:
    print("❌ 审计中心未检测到有效的样本输入，算法终止。")

print("\n🏁 Day08 列表核心案例全攻克，版本控制准备就绪！")
```

## 今日收获

**36 数据容器概述**：

数据容器就是一种可以容纳多份数据的数据类型（容器），每一个元素可以是任意类型的数据，如：字符串、数字、布尔等。

常见的数据容器有：列表(list)、字符串(str)、元组(tuple)、集合(set)、字典(dist)等。

**37 列表介绍**：

列表是数据容器中的一类，是一次性可以存储多个数据（元素）的，格式如：`s = [1, 2, 3, 4, 5, 6]`。其特点如下：

1、可以**存储不同类型的元素**；

2、元素**有序**、**可重复**、元素**可修改**。

通过索引（下标）来获取列表中的某个元素。索引起始值为0，表示列表的第1个元素。同时也存在反向索引，如：对于`s = [1, 2, 3, 4, 5, 6]`，`s[-1]`就表示列表中的最后一个元素，也就是6，反向索引的起始值为-1。

容器中的**元素按特定顺序排列**的，**可通过索引访问**的容器类型称之为**序列**。

**38 列表切片**：

列表切片：对操作的数据**截取其中一部分**的操作。**列表、字符串、元组（所有的序列类型的数据容器）**都**支持切片**操作。

切片的**语法**为：`序列数据[开始索引:结束索引:步长]`。

**开始索引**若未指定则**默认为0**；**结束索引**若未指定则默认为**列表长度**（即直到列表末尾）；**步长**若未指定则默认为**1**。索引采用正向反向均可。

**快速计算正负向索引值**的方法：以列表长为10为例，若正向索引为2，则反向索引绝对值为10-2=8，故对应的反向索引值为-8，二者都指代列表的第3个元素。

**39 列表方法**：

方法就是**物品（对象）**具备的**能力或功能（方法）**，**方法归属于对象**。

列表的常见方法就是指列表这种数据类型**内置的常见功能**（如**增、删、排序**等），常见的方法如下：

**append()**：在列表**尾部追加**元素，如：s.append(1)

**insert()**：在**指定索引前插入**元素，如：s.insert(0,92)

**remove()**：**移除**列表中**第一个**匹配到的值，如：s.remove(75)

**pop()**：**删除**列表中**指定索引**位置的元素，未指定索引时**默认为最后一个**，如：s.pop(7)

**sort()**：对列表中的值进行**从小到大排序**（仅限于列表中**所有元素数据类型一致**时进行），如：s.sort()

**reverse()**：**反转**列表元素，如：s.reverse()

**40 列表案例1**：

**sum()**是**求和**函数，用以获取列表**元素之和**；

**len()**是**获取列表长度**函数，用以获取列表**元素个数**。

**41 列表案例2**：

**in**用于**判断元素是否存在于列表中**，如果存在则返回True，不存在则返回False，，语法为：**元素 in 列表**，例如：

```python
if a not in new_list:   # 判断元素是否存在于列表中，如果存在则返回True，不存在则返回False
    new_list.append(a)
```

**解包**：将列表这一类容器**解开成一个个独立的元素**，运用格式为：*num_list1，即将num_list1容器解开成一个个独立的元素。

**快速合并两个列表**的方法：**解包**和**加法**。

**42 列表案例3**：

**列表推导式**就是**按照一定规则快速生成一个列表**的方法，其语法格式为：

1、**[要插入的值 for i in 序列/列表]**，如：`s = [i**2 for i in range(1,21)]`，意思是将每次i的平方存入到列表中

2、**[要插入的值 for i in 序列/列表 if 条件]**，如：`new_list = [num**2 for num in num_list if num % 2 == 0]`，意思是将其中偶数的平方存入列表

## 遇到的问题/注意事项

注意：如果**指定的索引超出范围则会报错**：list assignment index out of range。

注意：**切片不打印结束索引代指的元素**。

由于开始索引的默认值为0，步长默认值为1，结束索引默认值为列表长度，所以当**某个指定值等于默认值时可以省略**，但**第一个冒号不允许省略**。

两种删除方法的对比：**pop()**用于**按索引值删除**元素；而**remove()**用于**按元素值删除**元素。
