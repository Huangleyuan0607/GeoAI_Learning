## 今日收获

**18 算术运算符：**

**“ // ”**：表示**整除**，如10 // 3 = 3，整除结果为整数；

**“ % ”**：表示**取余/求模**，如10 % 3 = 1（10除以3取余数）；

“ ** ”：表示**幂指数**，如10 ** 3 = 1000（10的3次方）。

算术运算符**优先级**：** --> * / // % --> + -（**幂指数大于乘、除、整除、取余大于加、减**）。

**19 赋值运算符：**

**" = "**：赋值，把右边的值赋给左边的变量，如x = 1 + 2，则x的值为3；

**" += "**：加法赋值，如**x += 2即 x = x + 2**；

**" -+ "**：减法赋值，如**x -= 2即 x = x - 2**；

**" *= "**：乘法赋值，如**x *= 2即 x = x * 2**；

**" /= "：**除法赋值，如**x /= 2即 x = x / 2**；

**" %= "**：取余赋值，如**x %= 2即 x = x % 2**；

**" //= "**：取整除赋值，如**x //= 2即 x = x // 2**。

**20 比较运算符：**

**" == "**：等于，如a == b判断a是否等于b，**返回值为布尔值（True/False）**；

**" != "**：不等于，如a != b判断a是否不等于b，**返回值为布尔值（True/False）**；

**" >= "**：大于等于，如a >= b判断a是否大于等于b，**返回值为布尔值（True/False）**；

**"<= "**：小于等于，如a <= b判断a是否小于等于b，**返回值为布尔值（True/False）**。

**21 逻辑运算符：**

**" and "**：逻辑与（并且），**同时成立才符合条件**（两边均为True结果才为True），**返回值为布尔值（True/False）**；

**" or "**：逻辑或（或者），至少有一个成立就符合条件（只要有一个True结果就为True），**返回值为布尔值（True/False）**；

**" not "**：逻辑非（取反），取反操作，**True变为False，False变为True**，，**返回值为布尔值（True/False）**。

**" num >= 10 and num <= 20 "**在Pycharm里可简化链式比较成**" 10 <= num <= 20 ”** 。

## 遇到的问题/注意事项

需要注意的是，**除法运算的结果为浮点数**，即使结果为整数时也要加上.0。此外，**当有浮点数参与到运算中时结果为小数**。

**浮点数运算的精度损失**：当计算0.5 - 0.4得到0.09999999999999998时，是由于计算机底层是基于**二进制**进行数据的存储与处理，**二进制无法准确表示所有小数**，因此涉及到浮点数的运算时**可能**会损失精度。

Python通过**缩进**来表示**代码的层级关系或代码的归属**，所以要严格注意缩进。if语句里面的代码块，需要在前方缩进空格（建议4个空格或直接Tab），通过缩进来描述代码的层级关系（归属）。
