## 第二阶段：理论学习（约 0.5 小时）

深入理解以下核心概念，并思考它们在 GeoAI（地理人工智能）和日常代码清洗中的重要应用：

1. **取余（`%`）与取整除（`//`）**：在遥感图像大矩阵处理或瓦片分割中极其关键。例如，将单张大影像裁剪为 `256x256` 的小图块时，通过行/列号对 256 取整或取余，可以直接锁定图块在原图中的相对空间网格坐标。
2. **逻辑运算符（`and` / `or` / `not`）**：用于构建多条件遥感空间过滤筛选。例如筛选满足条件：`"绿波段反射率 > 0.3"` **且（`and`）** `"近红外波段反射率 < 0.1"` 的像元（常用于水体掩膜提取）。
3. **`if` 条件分支的缩进（Indentation）**：Python 抛弃了传统的花括号 `{}`，完全依赖缩进控制代码块所属关系。在 `if` 条件判定后，属于该分支的代码必须保持相同缩进（通常为 4 个空格）。

### 实验任务

创建 `condition_judge.py` 文件。手工编写并运行以下代码，模拟 GeoAI 遥感图像云检测以及研究生初试成绩上线判定的场景：

Python

```
# -*- coding: utf-8 -*-
"""
程序名称：condition_judge.py
作者：黄乐源
日期：2026-07-01
描述：攻克四大运算符与if基础判断，模拟初试成绩判定与遥感像元过滤
"""

print("================= 任务 1：算术与赋值运算符（遥感网格计算模拟） =================")
# 假设一张遥感影像总共有 1025 列，计划裁剪成每 256 列为一个格网图块
total_columns = 1025
tile_size = 256

tiles_count = total_columns // tile_size  # 取整除
remaining_cols = total_columns % tile_size  # 取余数

print(f"整图可完整裁剪出：{tiles_count} 个 [{tile_size}x{tile_size}] 的标准图块")
print(f"边缘剩余不足一个图块的列数：{remaining_cols} 列")

# 复合赋值运算：迭代器递增模拟
epoch_counter = 0
epoch_counter += 1  # 相当于 epoch_counter = epoch_counter + 1
print(f"当前训练迭代轮次累加至：{epoch_counter}")


print("\n================= 任务 2：比较、逻辑运算符与 if 判断（复试线判定） =================")
# 动态获取用户输入的统考成绩
print("--- 华中师范大学资源与环境专业（GeoAI方向）初试复试线审核系统 ---")
input_score_str = input("请输入你的研究生入学考试初试总分: ")

# 吸取 Day03 避坑经验：必须显式转换类型后才能进行数值大小比较
my_score = float(input_score_str)

# 设定模拟复试线
total_score_line = 360.0
is_rank_top = True  # 模拟排名是否前列（布尔值）

# 组合多条件判定：总分过线 且 排名靠前
if my_score >= total_score_line and is_rank_top:
    print(" 🎉 恭喜！你的初试总分已达到复试线，且排名符合要求。")
    print(" 📢 请立刻着手准备专业课面试与 GeoAI 预研成果汇报 PPT！")

# 基础 if 单独控制（未过线提示）
if my_score < total_score_line:
    print(" 💡 提示：成绩未达该模拟基准线，建议核对官方单科线或查看调剂信息。")
```
