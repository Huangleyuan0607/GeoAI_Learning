## 第二阶段：理论学习（约 0.5 小时）

深入理解以下三个核心概念在日常编写脚本和未来 GeoAI 数据处理中的重要作用：

1. **`f-string`（格式化字符串字面量）**：以 `f` 或 `F` 开头的字符串，花括号 `{}` 内可以直接放入变量名或表达式。在地理数据批处理中，常用于动态生成带编号的影像输入/输出路径（例如：`f"data/image_{img_id}.tif"`）。
2. **`.format()` 方法**：另一种强大且灵活的格式化手段。在处理复杂的、需要重复复用变量的日志或学术报告字符串输出时非常有用。
3. **`input()` 函数**：标准的控制台输入函数。需要反复强调的是：**无论用户输入的是数字还是符号，其返回值一律是 `str` 类型**。在处理诸如“请输入迭代次数(Epoch)”等定量参数时，必须配合类型转换。

### 实验任务

创建 `student_info.py` 文件。手工编写并运行以下代码，实现动态输入姓名、专业，并输出规范的格式化介绍：

Python

```
# -*- coding: utf-8 -*-
"""
程序名称：student_info.py
作者：黄乐源
日期：2026-06-30
描述：掌握字符串的高级格式化输出，定制GeoAI科研名片
"""

print("================= 任务 1：科研个人信息采集 =================")
# 动态获取用户输入
name = input("请输入你的姓名: ")
major = input("请输入你的硕士专业: ")
research_focus = "Geographic Information Science (GIS) & Deep Learning"
advisor = "刘鹏程老师"

print("\n================= 任务 2：多版本格式化输出 =================")

# 版本 A：传统的 % 占位符控制（了解即可）
info_a = "【传统版】你好，我是%s。我即将在华中师范大学攻读%s专业。" % (name, major)
print(info_a)

# 版本 B：灵活的 .format() 映射
info_b = "【format版】导师：{} | 研究方向：{}".format(advisor, research_focus)
print(info_b)

# 版本 C：极简高效的 f-string（推荐科研及工程主用！）
info_c = f"【f-string版】科研名片：\n 👤 姓名：{name}\n 🎓 专业：{major}\n 👨‍🏫 导师：{advisor}\n 🎯 核心领域：{research_focus}"
print(info_c)


print("\n================= 任务 3：GeoAI 日志路径生成模拟 =================")
# 模拟深度学习训练中，动态输出每个 Epoch 的 Loss 日志与权重保存路径
epoch = 5
train_loss = 0.024561

# 使用 f-string 并在花括号内直接进行浮点数精度控制（保留4位小数）
print(f"[Epoch {epoch:03d}] 训练完成！当前 Loss: {train_loss:.4f}")
print(f"[Save] 权重文件已自动保存至：python_basic/Day03/weights/model_epoch_{epoch}.pth")
```
