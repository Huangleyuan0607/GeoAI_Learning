深入理解以下核心概念，并思考它们在 GeoAI 空间分析和模型训练中的重要应用：

- **`while` 循环的条件驱动性与机器学习收敛**：与固定次数的遍历不同，`while` 循环是由条件（True/False）驱动的。这与深度学习中的 **Early Stopping（提前停止策略）** 逻辑完全一致——在训练损失（Loss）降到目标阈值以下，或者连续N个 Epoch 精度不再提升时，程序通过判定条件主动跳出循环，防止模型过拟合。
- **死循环的产生原因与防范**：如果在循环体内忘记更新控制变量（步进表达式），导致判定条件永远为 True，就会引发死循环。在处理海量遥感地物要素时，死循环会导致内存溢出或卡死，因此必须确保循环内部有明确的趋向于 False 的出口。

#### 实验任务

创建 `guess_number.py` 文件。手工编写并运行以下代码，前半部分利用 `while` 循环模拟深度学习模型训练中的 Loss 迭代收敛与早期停止（Early Stopping）机制；后半部分利用 `while` 实现经典的智能猜数字游戏，掌握条件控制的完整闭环：  

```
# -*- coding: utf-8 -*-
"""
程序名称：guess_number.py
作者：黄乐源
日期：2026-07-03
描述：掌握 while 循环语法，模拟深度学习 Loss 早期停止机制与编写猜数字游戏
"""
import random

print("================= 任务 1：while 循环应用（机器学习 Loss 早期停止机制模拟） =================")
# 模拟深度学习训练中，当 Loss（损失值）下降到指定阈值时停止迭代（Early Stopping）
current_loss = 2.5    # 初始损失值
target_loss = 0.3     # 目标停止阈值
epoch = 1             # 当前迭代轮次（Epoch）

print(f"🚀 开始训练模型... 初始 Loss: {current_loss}, 目标 Loss: {target_loss}")

# 当 Loss 高于目标值时，循环继续训练
while current_loss > target_loss:
    # 模拟每个 Epoch 训练后 Loss 逐渐下降（随机递减 0.2 ~ 0.5 之间的值）
    loss_drop = random.uniform(0.2, 0.5)
    current_loss -= loss_drop
    
    # 打印当前轮次的训练日志（保留4位小数）
    print(f"🔹 Epoch [{epoch}]: 训练中... 当前 Loss 降至: {current_loss:.4f}")
    
    # 严格防范死循环：更新步进变量（轮次累加）
    epoch += 1

print(f"🎉 训练完成！模型在第 {epoch - 1} 轮成功收敛，最终 Loss: {current_loss:.4f}，触发 Early Stopping 机制。")


print("\n================= 任务 2：while 条件控制实战（智能猜数字游戏） =================")
# 结合 while 条件控制与随机数，实现游戏逻辑闭环
print("--- 欢迎来到 GeoAI 数字密码破解游戏 ---")
print("系统已在 1-100 之间随机生成了一个目标地理要素编号，请尝试破解它！")

# 1. 系统自动生成 1-100 的随机数作为正确答案
target_num = random.randint(1, 100)

# 2. 初始化用户的猜测次数和当前的猜测状态
guess_count = 0
is_guessed = False

# 3. 使用 while 循环控制游戏，直到用户猜中才结束
while not is_guessed:
    input_str = input("请输入你猜测的编号 (1-100): ").strip()
    
    # 简单防错提示
    if not input_str.isdigit():
        print("⚠️ 输入无效！请输入纯数字。")
        continue
        
    guess_val = int(input_str)
    guess_count += 1
    
    # 多分支条件嵌套判断
    if guess_val > target_num:
        print("📈 提示：猜大了！该要素编号位于更低区间。")
    elif guess_val < target_num:
        print("📉 提示：猜小了！该要素编号位于更高区间。")
    else:
        print(f"🎯 恭喜你！成功破解密码！正确编号正是 【{target_num}】")
        print(f"📊 总共尝试次数：{guess_count} 次。")
        is_guessed = True  # 改变循环控制变量，优雅退出循环
```
