## 第二阶段：理论学习（约 0.5 小时）

深入理解以下四大基础数据类型及其在 GeoAI（地理人工智能）中的潜在应用场景：

1. **`int`（整型）**：整数。在遥感/GIS 中常用于表示**像素坐标**、**栅格像元值（如分类标签）**、**要素的 ID** 等。
2. **`float`（浮点型）**：小数。在地理计算中无处不在，例如**经纬度坐标**、**高程值（DEM）**、**遥感影像的反射率**等。
3. **`str`（字符串）**：文本。用单引号或双引号括起来。常用于表示**空间要素的名称**、**文件路径**、**遥感卫星传感器类型**等。
4. **`bool`（布尔型）**：真假值。只有 `True` 和 `False`。常用于**条件判断**或**掩膜（Mask）操作（如：某个像元是否为建筑物）**。

### 实验任务

创建 `variables_demo.py` 文件，手工编写并运行以下代码。通过 `input()` 动态获取用户输入，利用 `type()` 函数查看数据类型：

Python

```
# -*- coding: utf-8 -*-
"""
程序名称：variables_demo.py
作者：黄乐源
日期：2026-06-29
描述：探索Python基础数据类型与变量交互
"""

print("--- 任务 1：变量定义与数据类型查看 ---")
# 动态输入个人基本信息
name = input("请输入你的姓名: ")
age_str = input("请输入你的年龄: ")
gpa_str = input("请输入你的本科GPA: ")

# 注意：input() 接收的全部是字符串类型，需要进行显式类型转换
age = int(age_str)
gpa = float(gpa_str)
is_student = True  # 布尔型

# 打印变量及其所属的数据类型
print("姓名:", name, " | 类型:", type(name))
print("年龄:", age, " | 类型:", type(age))
print("GPA :", gpa, " | 类型:", type(gpa))
print("是否在读:", is_student, " | 类型:", type(is_student))


print("\n--- 任务 2：GeoAI 场景模拟之变量交换 ---")
# 模拟遥感影像处理中，由于通道调整需要交换波段编号（例如红波段与近红外波段）
band_red = 3
band_nir = 4
print("交换前 -> 红波段编号:", band_red, "，近红外波段编号:", band_nir)

# Python 特色的优雅变量交换
band_red, band_nir = band_nir, band_red
print("交换后 -> 红波段编号:", band_red, "，近红外波段编号:", band_nir)
```

**09 字面量类型：**

布尔值本质就是数字类型。在遇到数学运算时，True视为1 ，False视为0。

**10 变量类型：**

Python是**动态类型语言**，变量类型可以在程序运行过程中改变，即一个变量可以接受不同类型的值。（但在实际项目中最好还是只赋予和接受一种变量类型）

**11 标识符：**

标识符即元素（如函数、变量、类等）的名称。**命名规则**遵循：

只能包含字母、数字和下划线；

不能以数字开头；

不能使用关键字（如True,False等）；

严格区分大小写（如Age、age、AGE是三个变量）。

**命名规范**（变量）遵循：

见名知意；多个部分使用下划线连接；英文字母全小写。

**12变量案例(变量交换)：**

交换变量值需引入一个中间变量。

**13 数据类型：**

使用**type()**：查看指定数据的类型

使用**isinstance(数据,类型)**：查看指定数据的类型是否为指定类型，返回bool值（是：返回True；否：返回False）
